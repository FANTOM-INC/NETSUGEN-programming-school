# 使い方

## excel.py

excelを読み込み、データを取得したり書き込んだりします。

    python excel.py

## `find_file.py`

file, pdf, excel, wordの文章を検索して、与えた文字列が含まれているファイルを表示します。

l92-93を適宜変更してから実行

    python find_file.py
